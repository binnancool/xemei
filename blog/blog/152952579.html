Linux介绍

Linux的发行版有很多，主流的Linux有Asianux ，B2D Linux ，Conectiva Linux ，Debian GNU/Linux ，Fedora Core ，Gentoo Linux ，Knoppix Linux ，Linux From Scratch ，Magic Linux ，Mandriva Linux ，Red Hat Linux ，Slackware Linux ，Stanix Live CD ，SUSE Linux ，Turbo Linux ，Ubuntu Linux等，国内著名的Linux是红旗linux。

Redhat的系列有Redhat Linux、Fedora Core和Redhat Enterprise Linux。其中Redhat Linux的最高版本为9.0，停止发行新的版本；Fedora Core为免费版本，最高版本为11；Redhat Enterprise Linux为商业版本，需要付费使用。

在http://torrent.fedoraproject.org/ 能看到各种版本的Fedora 11，该怎样选择呢？

DVD or CD，这个不需要仔细说明了吧， n张碟和一张碟的区别。

ppc是IBM® POWER Architecture 体系， 也就是Power PC了。

x86或80x86是Intel首先开发制造的一种微处理器体系结构的泛称，包括Intel 8086、80186、80286、80386以及80486，因此其架构被称为“x86”。由於数字不能做注册商标，因此Intel及其AMD均在新一代processor采用可注册名称如Pentium。现在Intel把x86-32称为IA-32(Intel Architecture, 32-bit)。

i386， 是指兼容Intel 80386处理器, 适用于所有32bit cpu。

i686， 支持的cpu从Pentium 2 (686)开始,之前的型号不支持。

x86_64，AMD64 是64bit的processor架构，基于32bit的x86架构开发. 据说现64位的是试验性了，问题多，软件也少，性能也没多大提高。

KDE是指采用KDE desktop的，默认的则使用GNOME。

Live的版本只是然你试用一下的版本，大概就是一个光盘版的fedora 10， 系统都在光盘上。

综上选择i386就可以了, 至于DVD/CD， KDE/GNOME 就看自己的偏好了。
