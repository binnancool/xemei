Redhat 5.5 安装配置 KVM
1、KVM 需要有 CPU 的支持（Intel vmx 或 AMD svm），在安装 KVM 之前检查一下 CPU 是否提供了虚拟技术的支持
# egrep '^flags.*(vmx|svm)' /proc/cpuinfo
有显示说明CPU支持VT功能

2、在主板BIOS中开启CPU的Virtual Technolege(VT，虚化技术)

3、安装kvm
# yum install kvm virt-* libvirt

4、检查kvm模块是否安装
# lsmod | grep kvm

5、启动服务
# service libvirtd start

6、启动界面管理
# virt-manager
