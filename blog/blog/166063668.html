SLEE+UMG+HMP+软电话的安装与配置

一、slee（Linux版）的安装： （需要gcc的支持）

1、 安装文件存放目录：/SLEE(包括：setup_slee.jar、slee.tar、slee_setup.sh)

2、 cd /SLEE；sh slee_setup.sh（运行安装包）

3、 slee启动

4、 启动成功的标示：

5、 sleedown关闭

6、 把usml文件放在目录：/SLEE/ChannelSoft/CsCCP/SoftSwitch/Document下

7、 配置文件ATSEConfigData.cfg在目录/SLEE/ChannelSoft/CsCCP/SoftSwitch/ Config下。需要修改：

(1)

<AttributeSet>

<UVMGServer IP=”192.168.150.1” Port=”12000” UseVGProxy=”true” />UMG网关所在的IP地址和端口号

</AttributeSet>

(2) 其中修改XMLFile="业务实现.usml需要重启slee服务

<ResGroup Desc="服务资源组1" Type = "inbound">

     <DNList>

          <ComplexDN StartDN="5001" EndDN="5001"/>

     </DNList>

     <RunTime>

        <RunPeriod Begin="0" End="23">

             <Application Name="ATS服务" XMLFile="业务实现.usml" IsInbound="true" IsAutoStart="false"/>

          </RunPeriod>

        </RunTime>

    </ResGroup>

错误一：

运行slee时，报错：错误: 代理抛出异常 : java.net.MalformedURLException: Local host name unknown: java.net.UnknownHostException: bogon: bogon

修改文件/etc下的hosts文件，加入语句：192.168.150.128 bogon（其中192.168.150.128为eth0的IP地址，bogon为主机地址。）

不知怎么破坏了hosts文件。

8、 gcc的安装

1） 在安装系统时，在以下步骤如下设置：

A选择“现在定制”，如下图：

B选择“开发”----“开发工具”，在“开发工具”中自动安装gcc等

2） 在系统安装后，选择“应用程序”——“添加/删除软件”，在打开的图形界面中，如上图所示，选择“开发”----“开发工具”，在“开发工具”中自动安装gcc等。

3） 打开终端，输入命令yum install gcc，自动在网络上下载想依赖的软件包，自动安装。

二、UMG的安装

1、 放在目录D:\ChannelSoft下

2、 修改配置文件uvmg.ini中：

host_ip = 10.130.28.49（UMG所在的电脑IP地址）

       orig_path = D:\ChannelSoft\CsCCP\SoftSwitch\Document

dest_path = D:\ChannelSoft\CsCCP\SoftSwitch\Document

3、 声音文件放在D:\ChannelSoft\CsCCP\SoftSwitch\Document\Voice下

4、 运行UVMG_AzaleaD.exe

5、 运行成功的界面：其中“A New SLEE with IP=192.168.150.128 !!”代表在IP地址192.168.150.128上的SLEE启动。
