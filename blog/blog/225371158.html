把Tomcat6封装为服务

1、修改/bin目录下的service.bat文件的如下内容：

rem Set default Service name
set SERVICE_NAME=inform                          //服务名称
set PR_DISPLAYNAME=inform                    //显示名称

2、打开cmd，运行

安装服务：service install inform(到Tomcat的bin目录下执行)

删除服务：service remove inform(到Tomcat的bin目录下执行)

打开服务：net start inform

关闭服务：net stop inform

查询服务状态：sc query inform

设置为开机自动启动：sc config inform start= auto

删除服务：sc delete inform

3、查看Windows任务管理器——进程，显示tomcat6.exe

原因是tomcat的/bin目录下的tomcat6.exe,如果想改变进程tomcat6.exe，需要拷贝tomcat6.exe，修改为inform.exe。并修改/bin目录下service.bat中的tomcat6.exe，修改为inform.exe
