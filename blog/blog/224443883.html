Jconsole监控java虚拟机配置

1、本地监控

查找java.exe进程的进程号：xxxx

运行：jconsole xxxx

2、远程监控tomcat

在tomcat的bin/catalina.sh中的最上面写入：

set JAVA_OPTS=-Xms256m -Xmx512m -XX:PermSize=56M -XX:MaxNewSize=128m -XX:MaxPermSize=128m -Dcom.sun.management.jmxremote.port=9088 -Dcom.sun.management.jmxremote.ssl=false -Dcom.sun.management.jmxremote.authenticate=false

运行：jcosole，弹出如下图：IP：9088（上面黑体设置），加远程计算机的用户名和密码

3、java命令行

java -Dcom.sun.management.jmxremote.port=9088 -Dcom.sun.management.jmxremote.ssl=false -Dcom.sun.management.jmxremote.authenticate=false -classpath com.test
