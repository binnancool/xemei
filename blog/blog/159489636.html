VC++6.0项目在Visual Studio 8.0中打开，遇到的问题及修改方法

1、错误一：e:\dd\ccpreuseclass.h(14) : fatal error C1083: 无法打开包括文件:“fstream.h”: No such file or directory

修改方法：把ccpreuseclass.h中的#include <fstream.h>修改为#include <fstream>


2、错误二：e:\dd\sgip.cpp(301) : error C2065: “cerr”: 未声明的标识符

修改方法：在头文件：sgip.h中加上
               #include <iostream>
               using namespace std;


3、错误三：LINK : fatal error LNK1104: 无法打开文件“libcimt.lib”


修改方法：在C盘搜索文件libcimt.lib，在目录C:\Program Files\Microsoft SDKs\Windows\v5.0\Lib\IA64找到文件，复制到D:\Program Files\Microsoft Visual Studio 9.0\VC\lib文件夹下。

注：e:\dd是项目的存放路径；D:\Program Files\Microsoft Visual Studio 9.0是Visual Studio的安装目录。
