USStudio使用总结

一、RecordVoice节点的用法

1、 自定义知识变量

变量名：m_RecordVoice

类型：PronouncingLanguage

初始值：voice.wav

注：初始值一定要设置，因为录音保存在此文件中，文件名自己取，扩展名为.wav；扩展名根据底层设置而定；不设置初始值会提示错误：Invalid File；

2、 计算节点的VBScript脚本语言编程。

1） 对于SLEE内置对象的引用非常重要，参考文档：《SLEEJ内置组件产品规格说明书》

举例：调用HTTP请求组件：QNNetCOM，代码如下:

Dim Obj

Set Obj = CreateObject("QNNetCOM.clsHttp")//加载组件其中的类

If

//调用类clsHttp的方法HTTPRequest

Obj.HTTPRequest("http://192.168.150.1:8080/ChaFen/Search.jsp?number=d&password=d", 0, sResult, sDesc) Then

     MsgBox sResult//如果正确，则返回网页显示的内容，解析自己需要的信息。

// http://192.168.150.1:8080/ChaFen/Search.jsp?number=d&password=d是提交请求的处理页面，用get提交时可以显示的全部路径。

Else

     MsgBox sDesc

End If
