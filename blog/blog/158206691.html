Windows系统Telnet访问VMware Workstation虚拟机上的CentOS系统

1、CentOS系统中，默认安装了telnet的客户端，但是没有安装telnet的服务器端。因此首先要做的是安装telnet服务器，我首先想到的是在网上下载了telnet-server-0.17-38.el5.i386，但是在安装过程中需要安装xinetd，于是我找到其官方网站：http://www.xinetd.org下载；结果又出了问题，提示错误：no acceptable c compiler found in $PATH,我有到gcc的官方网站：http://gcc.gnu.org下载gcc-4.4.0版本，安装中又出现了错误。无奈………………

2、请教高手后：

1）安装xinetd命令：yum install xinetd

2）安装telnet-server命令：yum install telnet-server

以上两步也可以在见面上操作：应用程序----添加/删除程序，出现了“软件包管理者”，选择“列表”，从中选择要安装的telnet和xinetd，“应用”即可自动安装。

3）service xinetd start（telnet需要此服务，启动此服务就可以使用telnet命令.停止的话为stop）

4）service telnet start(此命令错误，telnet不是可标示的服务)

启动服务。

5）可以通过chkconfig设置服务启动、停止在各个执行等级的选项，设置Linux系统下次启动时服务是否启动，但是不实际的马上启动或停止服务。

设置方法：chkconfig 服务名 on/off

显示服务状态：chkconfig --list 服务名
