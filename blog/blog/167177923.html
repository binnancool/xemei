Html与jsp网页中书签的使用

当网页内容太长或要导航到其它页面的一个具体位置时，可以使用 Html 或Jsp中书签。
1.同一个web页面链接

首先设计书签

<a name=”top”></a>


在不同位置定义

<a href=”#top”>返回到top位置</a>


2.不同的页面之间转接

如果是在不同的页面之间链接时要在href属性中加上网页名称。

First.html或者First.jsp中定义了书签

<a name=”top”>Top的位置</a>


现在要从Second.html或者Second.jsp中转入First.html或者First.jsp并且将位置定到top所在的位置

可以设置

<a href=”first.html#top”>return first Top</a>

如果是按钮的submit，可以写JavaScript脚本：

<script type="text/javascript" language="javascript">
function ToTop()
{
window.location.href="First.jsp#Top";
}

</script>

<input type="button" value=" 提交 "
    onclick="ToTop()" style="BACKGROUND-COLOR: #ffffff">
